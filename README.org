* Dotfiles

A collection of configuration files managed with GNU Stow.

** Prerequisites

See =deps.txt= for required tools. [[https://www.gnu.org/software/stow/][GNU Stow]] is the only hard dependency.

*** Gnu Stow

GNU Stow is a symlink farm manager. See the [[https://www.gnu.org/software/stow/][official docs]] for details.

** Usage

A =.stowrc= file at the root configures stow defaults:

#+begin_src sh
  --target=~
  --dir=~/.dotfiles
  --no-folding
#+end_src

- =--target=~= - symlinks are created in =~=
- =--dir=~/.dotfiles= - package root is =~/.dotfiles=
- =--no-folding= - real directories are always created, never directory symlinks. Required because multiple packages contribute to shared directories like =~/.config/= and =~/.local/bin/=

Stow picks up =.stowrc= automatically when invoked from =~/.dotfiles=. To make it work from *any* directory, symlink it to =~=:

#+begin_src sh
  ln -s ~/.dotfiles/.stowrc ~/.stowrc
#+end_src

To stow a package:

#+begin_src sh
  stow emacs
#+end_src

To stow all packages at once:

#+begin_src sh
  cd ~/.dotfiles && stow */
#+end_src

Each package's directory structure is mirrored as symlinks in =~=.

** Homebrew

macOS packages are managed with [[https://brew.sh/][Homebrew]]. The =homebrew= package provides:

- =.config/homebrew/Brewfile= — package list tracked via =brew bundle=
- =.config/brew-aliases/= — native Homebrew alias scripts (e.g. =brew i=, =brew ud=)
- =.local/bin/brew-visit= — external command to open a package's homepage in the browser

*** Brewfile

After stowing the =homebrew= package, =~/.config/homebrew/Brewfile= is a symlink back into this repo. =brew bundle --global= resolves the Brewfile via =$XDG_CONFIG_HOME= (set to =~/.config= in =.bashrc=), so no extra configuration is needed.

Dump currently installed packages:

#+begin_src sh
  brew bundle dump --global --force
#+end_src

Because =~/.config/homebrew/Brewfile= is a symlink, the dump writes directly to the tracked file. Review and commit the diff:

#+begin_src sh
  git diff homebrew/.config/homebrew/Brewfile
  git add homebrew/.config/homebrew/Brewfile && git commit
#+end_src

To install all packages on a new system:

#+begin_src sh
  brew bundle --global
#+end_src

See the [[https://docs.brew.sh/Brew-Bundle-and-Brewfile][brew bundle docs]] for details.

*** Brew aliases

Homebrew natively reads =~/.config/brew-aliases/= for alias scripts. The files in this repo are stowed there and become available as =brew <alias>=. Run =brew alias= to see the current list.

*** brew visit

=brew visit <package>= opens the first URL from =brew info <package>= in your browser. It is an [[https://docs.brew.sh/External-Commands][external brew command]] picked up automatically from =~/.local/bin/= (which is in =PATH= via =macos.sh=).
